## PoeTest

This repo is a general framework for writing contracts and tests. It is a work in progress. This is provided as is.

### Makefile

See test/PoeUtils/BeforeAfter.sol after running command.

```shell
$ make interface
```


### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Format

```shell
$ forge fmt
```

### Help

```shell
$ forge --help
$ anvil --help
$ cast --help
```
